Dockerfile

Here's the Dockerfile used to create this custom image:

# Use the official Nginx Alpine image
FROM nginx:alpine

# Copy our custom HTML file to replace the default Nginx page
COPY index.html /usr/share/nginx/html/

# Expose port 80 for web traffic
EXPOSE 80

# Start Nginx when the container runs
CMD ["nginx", "-g", "daemon off;"]

Build & Run Commands

To build and run this Docker image:

# Build the image with a tag
docker build -t custom-nginx .

# Run the container, mapping port 80
docker run -d -p 8080:80 --name my-container custom-nginx

# View the running containers
docker ps

# Stop the container
docker stop my-container
